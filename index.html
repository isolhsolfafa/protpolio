Case Studies

프로젝트 성과

01
데이터 품질 혁신

QR/현장 로그 결측률 30~40%에서 품질 규칙 설계 → 정규화 → ETL 자동화를 거쳐 0.5% 미만으로 개선. 월간 보고서 2일 → 실시간 대시보드 전환. ROI 7,048% 달성.

02
PDA → Mobile App 전환

Google Sheets 수기 입력을 Flutter PWA + QR 스캔 즉시 기록으로 전환. DB Connection Pool로 응답 15~45s → <500ms. 현재 v2.2.0, Sprint 40+, 889 테스트 통과, Regression 0건 유지.

03
경영진 대시보드 구축

React + TypeScript + Vite SPA로 18개 페이지 개발. 공장현황 · 협력사 · 생산실적 · 불량 · CT분석 · 근태 등. shadcn/ui + Tailwind v4 + GxColors 디자인 시스템 통일.

04
AI 협업 개발 모델 실증

"기획=사람, 구현=AI" 분업 체계를 정립하여 6주 만에 31 Sprint · 18 릴리즈 달성. 1인 = 6~7명 SI 대체 실증. 비용 $110/월.

05
브랜드 & 디자인 시스템

G-AXIS 네이밍부터 로고, GxColors 컬러 토큰, DM Sans + JetBrains Mono 타이포, 컴포넌트 가이드까지 직접 설계. Flutter + React 전체에 일관된 디자인 시스템 적용.
